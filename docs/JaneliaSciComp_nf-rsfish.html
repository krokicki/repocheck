Setup Information

Prerequisites:

Nextflow is required to run this pipeline. [https://www.nextflow.io/docs/latest/getstarted.html]

Either Docker or Singularity is required to run the pipeline. [https://sylabs.io/guides/3.7/admin-guide/installation.html]

Setup Steps:

Install Nextflow using a curl command.

curl -s https://get.nextflow.io | bash

Install Nextflow using conda.

conda create --name nextflow -c bioconda nextflow

Install Singularity on CentOS Linux.

sudo yum install singularity

Run the RS-FISH pipeline with a singularity profile and specified input.

nextflow run JaneliaSciComp/nf-rsfish --input_image /path/to/image.n5 --input_dataset /c0/s0 --outdir ./output -profile singularity --rsfish_opts "--anisotropy 0.53 --minIntensity 5401 --maxIntensity 61053"
