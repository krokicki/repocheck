Setup Information

Prerequisites:

Make sure to unzip the h5 files and XML before using them. []

Setup Steps:

Run the Gala segmentation pipeline to obtain an oversegmented volume and prediction file.

gala-segmentation-pipeline -I '|graymaps|/*.png' --ilp-file |boundary classifier| --enable-gen-supervoxels --enable-gen-agglomeration --enable-gen-pixel --seed-size 5 |output directory| --segmentation-thresholds 0.0

Train a supervoxel boundary classifier for agglomeration using the oversegmented labels, prediction, and ground truth.

neuroproof_graph_learn |oversegmented_labels| |prediction| |groundtruth| --num_iterations 1

Use the trained agglomeration classifier to segment a new dataset.

neuroproof_graph_predict |oversegmented_labels| |prediction| |classifier|

Analyze the uncertainty of the resulting graph from the segmentation.

neuroproof_graph_analyze -g 1 -b 1
