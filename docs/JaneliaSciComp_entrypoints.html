Setup Information

Prerequisites:

Setup Steps:

Set the working directory to app and install the entrypoint binary.

ARG ENTRYPOINTS_VERSION=latest

Run curl to download and extract the entrypoint binary.

RUN curl -sL https://github.com/JaneliaSciComp/entrypoints/releases/download/${ENTRYPOINTS_VERSION}/entrypoints_linux_x86_64.tar.gz | tar -xz -C /app

Set the entrypoint for the container to the entrypoint binary.

ENTRYPOINT [ "/app/entrypoints", "-s", "/app/scripts/cmd" ]

Set the working directory to app and install the entrypoint binary in a multi-stage build.

FROM janeliascicomp/common-tools:1.0.2 as entrypoints-builder

Run curl to download and extract the entrypoint binary in a multi-stage build.

RUN curl -sL https://github.com/JaneliaSciComp/entrypoints/releases/download/${ENTRYPOINTS_VERSION}/entrypoints_linux_x86_64.tar.gz | tar -xz -C /tmp

Copy the entrypoint binary from the builder stage to the final container.

COPY --from=entrypoints-builder --chown=root:root /tmp/entrypoints /app/entrypoints

Set the entrypoint for the final container to the entrypoint binary.

ENTRYPOINT [ "/app/entrypoints", "-s", "/app/scripts/cmd" ]

Install the Go binary.

go install
