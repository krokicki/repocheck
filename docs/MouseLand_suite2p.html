Setup Information

Prerequisites:

Anaconda distribution of Python [https://www.anaconda.com/download/]

Setup Steps:

Open an anaconda prompt / command prompt with conda for python 3 in the path

Open an anaconda prompt / command prompt with `conda` for **python 3** in the path

Create a new environment

conda create --name suite2p python=3.9

Activate the new environment

conda activate suite2p

Install the minimal version of suite2p

python -m pip install suite2p

Run suite2p

python -m suite2p

Run suite2p from the command line

suite2p --ops --db
