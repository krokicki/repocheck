Setup Information

Prerequisites:

C++ compiler such as g++ or clang. [https://gcc.gnu.org/]

Setup Steps:

Compile the C++ shared library

g++ -std=c++11 -O3 -fPIC -shared -I./include src/compress_segmentation.cc src/decompress_segmentation.cc -o compress_segmentation.so

Install the Python package using pip

pip install compressed-segmentation

Install dependencies for building from source

sudo apt-get install g++ python3-dev

Install numpy package for Python

pip install numpy

Install the package from source

python setup.py install
