Setup Information

Prerequisites:

NI-DAQmx driver installation (version 20.1 recommended) [https://www.ni.com/en-us/support/downloads/drivers/download.ni-daqmx.html#348669]

Setup Steps:

Add the NIDAQ.jl package

]add NIDAQ

Using NIDAQ.jl, query system properties

using NIDAQ

Get properties with no input arguments

getproperties()

Get properties for a specific device

getproperties("Dev1")

Add analog input channels

t = analog_input("Dev1/ai0:1")

Start data acquisition

start(t)

Read data from the acquisition

read(t, 10)

Stop the acquisition

stop(t)

Clear the task

clear(t)
