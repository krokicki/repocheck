Setup Information

Prerequisites:

Install conda for managing environments and dependencies [https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html]

Setup Steps:

Create the conda environment using the specified environment file

conda env create --file environment.yml

Activate the conda environment

conda activate n5-tools-dask

Create output directories for TIFFs and N5 files

mkdir output_tiffs output_n5

Convert N5 images to TIFF format

python src/n5_to_tif.py -i data/test.n5/ -d mri/c0/s0 -o output_tiffs

Convert TIFF files back to N5 format

python src/tif_to_n5.py -i output_tiffs -o output_n5 -d mri/s0 -c 64,64,64 --compression gzip

Add multiscale imagery to the N5 file

python src/n5_multiscale.py -i output_n5 -d mri
