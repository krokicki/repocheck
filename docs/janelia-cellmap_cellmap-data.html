Setup Information

Prerequisites:

Setup Steps:

Create a new micromamba environment named 'cellmap' with Python 3.11

micromamba create -n cellmap -y -c conda-forge -c pytorch python=3.11

Activate the 'cellmap' environment

micromamba activate cellmap

Clone the CellMap-Data repository from GitHub

git clone https://github.com/janelia-cellmap/cellmap-data.git

Change directory to the cloned repository

cd cellmap-data

Install the package in editable mode

pip install -e .
