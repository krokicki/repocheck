Setup Information

Prerequisites:

Python package for creating JSON Web Tokens [https://pyjwt.readthedocs.io/en/stable/]

Setup Steps:

Install PyJWT

pip install PyJWT

Create a JWT token

pyjwt --key=MYSECRET encode email=EMAILADDR level=admin|readwrite|readonly

Add a user with readonly permission to the API

curl -X POST -H "Content-Type: application/json" -H "Authorization: Bearer " https:///users -d '{"foo@bar": "admin"}'
