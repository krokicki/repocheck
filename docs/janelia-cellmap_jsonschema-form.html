Setup Information

Prerequisites:

Fly Command Line Interface [https://fly.io/docs/getting-started/installing-flyctl/]

Setup Steps:

Create a new Remix app using the Indie Stack template

npx create-remix@latest --template remix-run/indie-stack

Initialize a Remix project

npx remix init

Run initial setup commands

npm run setup

Start the development server

npm run dev

Sign up for Fly and log in

fly auth signup

Create a production Fly app

fly apps create jsonschema-form-76e1

Create a staging Fly app

fly apps create jsonschema-form-76e1-staging

Initialize a new Git repository

git init

Add a remote GitHub repository

git remote add origin

Set the SESSION_SECRET for production

fly secrets set SESSION_SECRET=$(openssl rand -hex 32) --app jsonschema-form-76e1

Set the SESSION_SECRET for staging

fly secrets set SESSION_SECRET=$(openssl rand -hex 32) --app jsonschema-form-76e1-staging

Create a persistent volume for the SQLite database in production

fly volumes create data --size 1 --app jsonschema-form-76e1

Create a persistent volume for the SQLite database in staging

fly volumes create data --size 1 --app jsonschema-form-76e1-staging
