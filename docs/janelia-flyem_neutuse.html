Setup Information

Prerequisites:

pyzem is required to run the database [https://github.com/janelia-flyem/pyzem]

Setup Steps:

Clone the neutuse repository

git clone https://github.com/janelia-flyem/neutuse.git

Navigate into the neutuse directory

cd neutuse

Install neutuse

python setup.py install

Clone the pyzem repository

git clone https://github.com/janelia-flyem/pyzem

Navigate into the pyzem directory

cd pyzem

Install pyzem

python setup.py install

Create configuration directory

mkdir neutuse-configs

Navigate into the configuration directory

cd neutuse-configs

Create skeletonize configuration file

cat > skeletonize_config.yaml << EOF number: 1 log: /tmp/neutuse_skeletonize.log command: /miniconda/envs/neutu/bin/neutu EOF

Create process configuration file

cat > process_config.yaml << EOF address: host: localhost port: 2018 number: 1 log: /tmp/neutuse.log process: - name: skeletonize config: skeletonize_config.yaml EOF

Start the database service

neutuse start database --config process_config.yaml

Start the polling process in a separate terminal

neutuse start process --config process_config.yaml
