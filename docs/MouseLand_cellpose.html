Setup Information

Prerequisites:

Python version must be 3.8 or later. [https://www.python.org/downloads/]

Setup Steps:

Create a new conda environment for Cellpose.

conda create --name cellpose python=3.10

Activate the cellpose conda environment.

conda activate cellpose

Install Cellpose with the GUI support.

python -m pip install cellpose[gui]

Run Cellpose using the command line interface.

python -m cellpose
