Setup Information

Prerequisites:

Micromamba for package management [https://mamba.readthedocs.io/en/latest/installation/micromamba-installation.html#automatic-install]

Setup Steps:

Install micromamba

"${SHELL}" <(curl -L micro.mamba.pm/install.sh)

Create a new environment with Python 3.11

micromamba create -n csc -y python=3.11

Activate the environment

micromamba activate csc

Clone the repository

git clone https://github.com/janelia-cellmap/cellmap-segmentation-challenge

Install the repository in editable mode

cd cellmap-segmentation-challenge

Install the package

pip install -e .

Fetch the challenge data

csc fetch-data

Train a 2D model

cd examples

Run the training script for 2D

python train_2D.py

Run TensorBoard to visualize training metrics

tensorboard --logdir=tensorboard
