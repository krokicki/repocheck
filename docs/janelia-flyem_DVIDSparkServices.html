Setup Information

Prerequisites:

Miniconda for managing packages and environments [http://conda.pydata.org/miniconda.html]

Setup Steps:

Install miniconda (Linux)

wget https://repo.continuum.io/miniconda/Miniconda-latest-Linux-x86_64.sh

Run the installer for miniconda

bash Miniconda-latest-Linux-x86_64.sh

Activate conda environment

CONDA_ROOT=`conda info --root`

Activate the root environment

source ${CONDA_ROOT}/bin/activate root

Create a new conda environment and install dvidsparkservices

conda create -n -c flyem dvidsparkservices

Build the package

python setup.py build

Install the package

python setup.py install

Run the workflow (as an example)

spark-submit --master local[4] DVIDSparkServices/workflow/launchworkflow.py ComputeGraph -c .json

Unzip test data for running integration tests

gunzip integration_tests/resources/*.gz

Run integration tests

python integration_tests/launch_tests.py
