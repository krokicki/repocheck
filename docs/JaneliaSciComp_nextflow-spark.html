Setup Information

Prerequisites:

Nextflow [https://www.nextflow.io]

Singularity or Docker [https://sylabs.io or https://www.docker.com]

Setup Steps:

Run the Spark application

./main.nf -profile lsf

Specify runtime options and other parameters

--runtime_opts "-e -B $PWD/local -B $PWD/examples -B /nrs/jacs/jacsData/filestore"

Specify LSF options

--lsf_opts "-P scicompsoft -W 30"

Set the number of workers

--workers 3

Set the number of cores per worker

--worker_cores 4

Set memory per core

--gb_per_core 15

Set driver cores

--driver_cores 1

Set driver memory

--driver_memory 10g

Set Spark working directory

--spark_work_dir "$PWD/local"

Set the path to the application JAR

--app $PWD/local/app.jar

Set the main class for the application

--app_main org.janelia.colormipsearch.cmd.SparkMainEntry

Set the application arguments

--app_args "searchFromJSON, -m $PWD/local/testData/mask-2.json, -i $PWD/local/testData/sgal4Targets.json, --mirrorMask, --maskThreshold 100, --dataThreshold 100, --xyShift 2, --pctPositivePixels 2.0, --pixColorFluctuation 2, -od $PWD/local/testData/cdsresults.test"
