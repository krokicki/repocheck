Setup Information

Prerequisites:

CUDA and CUDNN drivers from nvidia.com for GPU support [https://www.nvidia.com/en-us/deep-learning-ai/solutions/ gpu-cloud/]

Setup Steps:

Download the SongExplorer ZIP file for your operating system

$ unzip songexplorer--.zip

Create a folder for ground truth data

$ mkdir -p groundtruth-data/round1

Copy example WAV file into the created folder

$ cp PS_20130625111709_ch3.wav groundtruth-data/round1

Start SongExplorer's GUI

$ ./OPEN-WITH-TERMINAL.sh

Monitor training progress

$ watch tail trained-classifier1/train_1r.log

Delete old prediction files to save space

$ rm groundtruth-data/*/oldfiles*
