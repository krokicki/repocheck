Setup Information

Prerequisites:

Nextflow version 20.10.0 or greater [https://www.nextflow.io/docs/latest/getstarted.html]

Singularity version 3.5 or greater [https://sylabs.io/guides/3.7/admin-guide/installation.html]

Setup Steps:

Install Nextflow using curl

curl -s https://get.nextflow.io | bash

Clone the multifish repository

git clone https://github.com/JaneliaSciComp/nf-demos.git

Run the setup script to pull in external dependencies

./setup.sh

Launch a pipeline (example command)

./n5-tools-dask/tiff-to-n5.nf.nf [arguments]
