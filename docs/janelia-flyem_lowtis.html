Setup Information

Prerequisites:

Miniconda installation [http://conda.pydata.org/miniconda.html]

Setup Steps:

Create a build directory and navigate into it

mkdir build

Run cmake to configure the build system

cmake -DLIBDVID_INCLUDE=path_to_libdvid_include ..

Compile the project

make

Install the compiled project

make install

Create a new conda environment with lowtis

conda create -n CHOOSE_ENV_NAME -c flyem-forge lowtis

Activate the conda environment

source ${CONDA_ROOT}/bin/activate root
