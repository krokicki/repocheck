Setup Information

Prerequisites:

Conda package manager for installing dependencies [https://docs.conda.io/en/latest/miniconda.html]

Setup Steps:

Create a new Conda environment named 'video'

conda create --name video

Activate the Conda environment 'video'

conda activate video

Install the necessary dependencies in the activated environment

conda install -c conda-forge pillow ffmpeg opencv

Run the split-screen video command with input frames and output directory

python split-screen-video/splitScreenVideo.py -i1 /tmp/framesA -i2 /tmp/framesB -o /tmp/AB
