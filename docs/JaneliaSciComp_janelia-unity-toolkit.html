Setup Information

Prerequisites:

Unity version required to install packages. [https://docs.unity3d.com/Manual/upm-ui-local.html]

Setup Steps:

Clone the repository into a directory on your computer.

git clone https://github.com/JaneliaSciComp/janelia-unity-toolkit.git

Open Unity and choose Package Manager from the Window menu.

Choose 'Package Manager' from the Unity editor's 'Window' menu.

Add package from disk by selecting the package's directory.

Press the small '+' button in the upper-left corner of the Package Manager window, and click on the 'Add package from disk...' item.

Navigate to the package's directory and select package.json file to load the package.

Use the file chooser to navigate to a package's directory (e.g., org.janelia.package-installer) in the cloned janelia-unity-toolkit repository. Select the package.json file and press 'Open'.
