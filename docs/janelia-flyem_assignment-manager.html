Setup Information

Prerequisites:

Docker and Docker Compose installed on the server [https://docs.docker.com/get-docker/]

Setup Steps:

Build the Docker container

docker build --tag registry.int.janelia.org/flyem/assignment-manager .

Push the Docker image to the registry

docker push registry.int.janelia.org/flyem/assignment-manager

Create a new database instance

sudo mkdir /data/mysql/assignment

Change ownership of the database directory

sudo chown mysql:mysqldba /data/mysql/assignment

Start the database service using Docker Compose

docker-compose -f docker-compose-prod.yml up db

Navigate to the assignment responder directory

cd /opt/flask/assignment-responder

Stop any running Docker services related to the project

docker-compose -f docker-compose-prod.yml down

List all Docker images

docker image ls

Remove the specific Docker image for assignment-manager

docker image rm

Remove the static volume for assignment-manager

docker volume rm assignment-manager_static_volume

Pull the latest Docker image from the registry

docker pull registry.int.janelia.org/flyem/assignment-manager

Start the application using Docker Compose

docker-compose -f docker-compose-prod.yml up -d
