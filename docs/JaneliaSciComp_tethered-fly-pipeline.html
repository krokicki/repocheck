Setup Information

Prerequisites:

Nextflow (version 20.10.0 or greater) [https://www.nextflow.io/docs/latest/getstarted.html]

Singularity (version 3.5 or greater) [https://sylabs.io/guides/3.7/admin-guide/installation.html]

Setup Steps:

Install Nextflow

curl -s https://get.nextflow.io | bash

Add Nextflow to PATH

mv nextflow bin/

Update bashrc to include Nextflow in PATH

echo 'export PATH=$PATH:/$HOME/bin' >> .bashrc

Set Nextflow version to 20.10.0

export NXF_VER=20.10.0

Install Singularity

sudo yum install singularity

Clone the repository

git clone git@github.com:JaneliaSciComp/tethered-fly-pipeline.git

Run the pipeline locally on a Mac

./main.nf -profile localDocker

Run the pipeline locally on a Linux machine

./main.nf

Run the pipeline on Janelia's LSF cluster

bsub -e jobErrorLog.err -o jobOutputLog.out \ -P "+{LSF_PROJECT_CODE}" \ nextflow run main.nf \ -profile janeliaLSF \ --lsf_opts "-P ${LSF_PROJECT_CODE}" \
