Setup Information

Prerequisites:

Nextflow version 20.10.0 or greater [https://www.nextflow.io/docs/latest/getstarted.html]

Singularity version 3.5 or greater [https://sylabs.io/guides/3.7/admin-guide/installation.html]

Setup Steps:

Install Nextflow

curl -s https://get.nextflow.io | bash

Alternatively, install Nextflow as a conda package

conda create --name exm -c bioconda nextflow

Install Singularity on CentOS Linux

sudo yum install singularity

Clone the expansion microscopy pipeline repository

git clone https://github.com/JaneliaSciComp/expansion-microscopy-pipeline.git

Run setup to pull in external dependencies

./setup.sh

Launch the stitch pipeline

./stitch_pipeline.nf [arguments]

Run the main pipeline

./main.nf [arguments]
