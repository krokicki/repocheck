Setup Information

Prerequisites:

Python 3.5 or higher is required. []

Setup Steps:

Install the Gala package using pip.

pip install gala

Create a Conda environment for gala using the environment file.

conda env create -f environment.yml

Install Gala using distutils in your preferred Python environment.

python setup.py install

Build the project using CMake and make.

cmake -D BUILDEM_DIR=/path/to/platform-specific/build/dir

Compile the project.

make

Run tests to check the installation of Gala.

py.test
