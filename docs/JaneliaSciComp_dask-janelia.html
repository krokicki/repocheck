Setup Steps:

Set up a default Dask cluster (either LocalCluster or LSFCluster)

from dask_janelia import get_cluster

Create a Dask client to manage the cluster

cl = Client(cluster)

Set up a LocalCluster explicitly on the compute cluster

cluster = get_cluster(deployment='local')

Request more memory for a LSF job when setting up the cluster

cluster = get_cluster(lsf_kwargs={'mem': '32GB', 'project' : 'foo'})
