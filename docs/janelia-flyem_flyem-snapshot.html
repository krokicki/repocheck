Setup Information

Prerequisites:

Conda package manager to create and manage environments [https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html]

Setup Steps:

Create a new conda environment named 'flyem' with the neuclease package

conda create -n flyem neuclease

Clone the flyem-snapshot repository from GitHub

git clone ssh://git@github.com/janelia-flyem/flyem-snapshot

Change directory into the cloned repository

cd flyem-snapshot

Install the package in the current environment

python setup.py develop
