Setup Information

Prerequisites:

Python 3 is required to create a virtual environment and run the project. [https://www.python.org/downloads/release/python-360/]

Setup Steps:

Change directory to the project folder.

cd /opt/flask/es-responder

Create a virtual environment.

python3 -m venv myenv

Activate the virtual environment.

source myenv/bin/activate

Install the project dependencies from requirements.txt.

pip3 install -r requirements.txt

Start the server using the main script.

python3 es_responder.py
