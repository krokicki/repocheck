Setup Information

Prerequisites:

Singularity must be installed for this project to run containers [https://singularity.lbl.gov/docs-installation/]

Setup Steps:

Build one or more tools into Singularity containers

./manage.sh build [tool1] [tool2] [tool3] ...

Set the target location for Singularity deployment

export JACS_SINGULARITY_DIR=/groups/jacs/jacsDev/servers/jacs-data/executables/singularity

Deploy the built container to the defined target location

./manage.sh deploy [tool1] [tool2] [tool3] ...

Run the container with the specified application and bind external paths

singularity run -B /external/path1 -B /external/path2 --app appName container.img -i /external/path1 -o /external/path2
