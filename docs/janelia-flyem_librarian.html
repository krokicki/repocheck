Setup Information

Prerequisites:

Go programming language [http://golang.org]

Setup Steps:

Install the librarian project using Go get

go get github.com/janelia-flyem/librarian

Install the cron library using Go get

go get github.com/janelia-flyem/go/cron

Install the Goji library using Go get

go get github.com/zenazn/goji

Build the librarian project

go install github.com/janelia-flyem/librarian

Run the librarian server with a specified log file

librarian /path/to/librarian.log
