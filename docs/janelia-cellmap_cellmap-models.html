Setup Information

Prerequisites:

A conda or mamba environment for package installation [https://docs.anaconda.com/free/miniconda/#quick-command-line-install]

Setup Steps:

Create a new conda environment named 'cellmap' and install Python and PyTorch

conda env create -y -n cellmap python=3.10 pytorch -c pytorch

Activate the conda environment

conda activate cellmap

Install the cellmap-models package

pip install cellmap-models
