Setup Information

Prerequisites:

Python and Flask must be installed, along with MongoDB for the database. [https://www.mongodb.com/]

Setup Steps:

Run the update_dois.py script to pull and update DOIs.

my_venv/bin/python3 update_dois.py --manifold prod --write

To see help documentation for the update_dois.py script.

my_venv/bin/python3 update_dois.py --help
