Setup Information

Prerequisites:

Install Miniconda if it is not already installed. [https://docs.conda.io/en/latest/miniconda.html]

Setup Steps:

Clone the repository

git clone https://github.com/JaneliaSciComp/EM_recon_pipeline.git

Change directory to the cloned repository

cd EM_recon_pipeline

Create the conda environment from the environment file

conda env create -f janelia_emrp.environment.yml

Activate the conda environment

conda activate janelia_emrp

Install dependencies using poetry

poetry install
