Setup Information

Prerequisites:

Miniconda package manager [http://conda.pydata.org/miniconda.html]

Setup Steps:

Install Miniconda (Linux)

wget https://repo.continuum.io/miniconda/Miniconda-latest-Linux-x86_64.sh

Run Miniconda installer (Linux)

bash Miniconda-latest-Linux-x86_64.sh

Install Miniconda (Mac)

wget https://repo.continuum.io/miniconda/Miniconda-latest-MacOSX-x86_64.sh

Run Miniconda installer (Mac)

bash Miniconda-latest-MacOSX-x86_64.sh

Activate conda

CONDA_ROOT=`conda info --root`

Activate conda environment

source ${CONDA_ROOT}/bin/activate root

Create a conda environment for NeuroProof

conda create -n CHOOSE_ENV_NAME -c flyem neuroproof

Clone NeuroProof repository

git clone https://github.com/janelia-flyem/neuroproof

Change to the NeuroProof directory

cd neuroproof

Run configuration script for conda

./configure-for-conda.sh ${PREFIX}

Create build directory

cd build

Compile the project

make -j4

Install the compiled project

make install

Test the installation

make test
