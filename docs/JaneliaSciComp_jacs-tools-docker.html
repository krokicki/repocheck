Setup Steps:

Build the NeuronBridge aligner container

./manage.sh build neuronbridge-aligner

Run the NeuronBridge aligner with local data

localip=$(ifconfig en0 | grep inet | awk '$1=="inet" {print $2}')

Allow local host access to the container

xhost + $localip

Set the FB_MODE_PARAM environment variable

FB_MODE_PARAM="-e FB_MODE=false -e DISPLAY=$localip:0"

Set the local data directory

LOCAL_DATA_DIR=$PWD/local/testData

Set the local templates directory

LOCAL_TEMPLATES_DIR=$LOCAL_DATA_DIR/templates

Run the Docker container to execute the aligner script with defined parameters

docker run \ -v $LOCAL_DATA_DIR:/data \ -v $LOCAL_TEMPLATES_DIR:/alignment_templates \ -it \ registry.int.janelia.org/jacs-scripts/neuronbridge-aligner:1.0 \ /opt/aligner-scripts/run_aligner.sh \ -debug \ --xyres 1 \ --zres 1 \ --nslots 4 \ --nchannels 2 \ --templatedir ${TEMPLATES} \ -i $INPUT \ -o $OUTPUT \ $*
