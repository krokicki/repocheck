Setup Information

Prerequisites:

Conda package manager [https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html]

Setup Steps:

Create a conda environment with all the dependencies including Jupyter

conda env create -f environment.yml

Activate the conda environment

conda activate neuronbridge-python

Install the module as a Jupyter kernel

python -m ipykernel install --user --name=neuronbridge-python

Install the module for development

pip install -e .

Run the unit tests

pytest tests
