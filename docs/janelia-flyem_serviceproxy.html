Setup Information

Prerequisites:

Go programming language [https://golang.org/doc/install]

Setup Steps:

Install serviceproxy

go get github.com/janelia-flyem/serviceproxy

Change directory to the serviceproxy source

cd $GOPATH/src/github.com/janelia-flyem/serviceproxy

Install adderexample and serviceregister

go get ./...

Launch the service proxy server

serviceproxy [-port WEBPORT (default 15333)]

Launch the adderexample service

adderexample ADDR

Register a service called 'foo'

registerservice foo 15555 ADDR
