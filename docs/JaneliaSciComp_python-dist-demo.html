Setup Information

Prerequisites:

Python environment management tool for creating isolated environments. [https://docs.conda.io/en/latest/]

Setup Steps:

Create a new conda environment for the project.

$ conda create --name python-dist-demo-1 python=3.10

Activate the conda environment.

$ conda activate python-dist-demo-1

Install h5py package for handling HDF5 files.

$ conda install h5py

Install Nuitka for compiling the Python script.

$ python -m pip install nuitka

Change to the demo directory.

$ cd python-dist-demo/demo1

Compile the demo script using Nuitka with required flags.

$ python -m nuitka --onefile --include-data-files=../VERSION=VERSION demo1.py

Run the executable created by Nuitka.

$ ./demo1.bin -i path/to/example.h5j
