Setup Information

Prerequisites:

Install Miniconda [https://docs.conda.io/projects/conda/en/latest/user-guide/install/linux.html]

Clone pyktx [https://github.com/JaneliaSciComp/pyktx]

Setup Steps:

Set channel priority for conda

conda config --set channel_priority strict

Create conda environment from environment.yml

conda env create -f environment.yml

Activate the conda environment

conda activate octree

Install pyktx from the specified path

pip install /path/to/pyktx

Convert TIFF slices or N5/Zarr datasets to octree format

python tiff2octree.py -i /path/to/input_dir -o /path/to/output_dir -d 2ndmax -t 16 --ktx

Convert TIFF stacks to octree format

python tiff2octree.py -f /input_slices/tiff -o /output/octree -d 2ndmax -t 16 --ktx

Convert multi-channel images to octree format

python tiff2octree.py -i /path/to/input_dir/ch1,/path/to/input_dir/ch2 -o /output/octree/ -d 2ndmax -t 8 --ktx
