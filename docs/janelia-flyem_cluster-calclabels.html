Setup Information

Prerequisites:

Install Golang to use 'go get' command [https://golang.org/doc/install]

Setup Steps:

Install the cluster-calclabels service

go get github.com/janelia-flyem/cluster-calclabels

Install CalcLabelOrchestration

python setup.py build

Install CalcLabelOrchestration

python setup.py install
