Setup Information

Prerequisites:

Igor Pro 6 legacy ITC 16 XOP for ITC 16 [http://www.heka.com/instrutech/IgorXOPs/Win/ITC16_X86_XOP.zip]

Igor Pro 6 legacy ITC 18 XOP for ITC 18 [http://www.heka.com/instrutech/IgorXOPs/Win/ITC18_X86.zip]

Bruxton Corporation SIDX 7.2 XOP for Andor iXon Ultra camera imaging [http://www.bruxton.com/SIDX/index.html]

Setup Steps:

Unzip the DataPro release zip file

unzip DataPro-release_8.24.zip

Copy the unzipped DataPro folder to the Igor Pro User Files folder

cp -r DataPro-release_8.24 "C:/Path/To/Igor Pro 6 User Files/"

Add DataPro to the Igor Pro procedure file for each experiment

echo "#include ":DataPro-release_8.24:DataPro"" >> "C:/Path/To/Your/Experiment.pro"
