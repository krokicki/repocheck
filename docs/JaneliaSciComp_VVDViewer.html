Setup Information

Prerequisites:

Git [https://git-scm.com/]

CMake [http://www.cmake.org/]

wxWidgets [https://github.com/wxWidgets/wxWidgets]

Visual Studio [https://visualstudio.microsoft.com/]

Setup Steps:

Clone wxWidgets from the repository

git clone https://github.com/wxWidgets/wxWidgets.git

Navigate to wxWidgets directory

cd ~/wxWidgets/

Create a build directory

mkdir mybuild

Navigate to the build directory

cd mybuild

Configure wxWidgets

../configure --disable-shared --enable-macosx_arch=x86_64 --enable-unicode --with-cocoa --enable-debug --with-macosx-version-min=10.9 --enable-stl --enable-std_containers --enable-std_iostreams --enable-std_string --enable-std_string_conv_in_wxstring --with-libpng --with-libtiff --with-libjpeg

Build wxWidgets

make

Install wxWidgets

make install

Clone VVDViewer repository

git clone https://github.com/takashi310/VVD_Viewer.git

Create a build directory for VVDViewer

mkdir build

Navigate to the VVDViewer build directory

cd build

Generate Xcode project files for VVDViewer

cmake ../VVD_Viewer -G Xcode
