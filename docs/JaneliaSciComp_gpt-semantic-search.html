Setup Information

Prerequisites:

Docker must be installed [https://docs.docker.com/get-docker/]

Setup Steps:

Start the Compose deployment which runs Weaviate and Streamlit webapp

docker compose up -d

Create a virtual environment and install dependencies

virtualenv env

Activate the virtual environment

source env/bin/activate

Install required Python packages

pip install -r requirements.txt

Run the indexing for Slack export

./index_slack.py -i ./data/slack/slack_export_Janelia-Software_ALL -c Janelia

Run the indexing for Wiki export

./index_wiki.py -i ./data/wiki -c Janelia

Run the indexing for the Janelia website

./index_web.py -i ./data/janelia.org -c Janelia

Run the webapp in development mode using Streamlit

streamlit run 1_üîç_Search.py -- -w http://search.int.janelia.org:8777
