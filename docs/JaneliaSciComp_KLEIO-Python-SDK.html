Setup Steps:

Open a Zarr backed versioned file system store and prepare it for committing changes

index_store = ZarrIndexStore(INDEX_PATH)

Create a VersionedFSStore to handle versioning.

store = VersionedFSStore(index_store, RAW_PATH)

Open the Zarr store in append mode.

z = zarr.open(store, mode="a")

Commit the current changes to the version control.

store.vc.commit()

Push the changes to the version control repository.

store.vc.push()

Create a new branch in the version control system.

store.vc.create_new_branch(BRANCH_NAME)

Checkout to the newly created branch.

store.vc.create_new_branch(BRANCH_NAME)
