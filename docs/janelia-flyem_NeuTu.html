Setup Information

Prerequisites:

Miniconda installation is required before running these commands. [https://docs.conda.io/en/latest/miniconda.html]

Setup Steps:

Download and run the installation script for NeuTu.

curl -X GET https://raw.githubusercontent.com/janelia-flyem/NeuTu/develop/neurolabi/shell/setup_neutu.sh > setup_neutu.sh

Execute the downloaded setup script with the intended installation directory.

bash setup_neutu.sh neutu-develop

Install Miniconda on Mac.

curl -X GET https://repo.continuum.io/miniconda/Miniconda3-latest-MacOSX-x86_64.sh > Miniconda-latest-MacOSX-x86_64.sh

Run the Miniconda installer on Mac.

bash Miniconda-latest-MacOSX-x86_64.sh

Install Miniconda on Linux.

curl -X GET https://repo.continuum.io/miniconda/Miniconda3-latest-Linux-x86_64.sh > Miniconda-latest-Linux-x86_64.sh

Run the Miniconda installer on Linux.

bash Miniconda-latest-Linux-x86_64.sh

Activate the conda environment after installation.

source /bin/activate

Launch the NeuTu application after successful installation.

neutu
