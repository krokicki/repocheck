Setup Information

Prerequisites:

Nextflow version 20.10.0 or greater [https://www.nextflow.io]

Singularity version 3.5 or greater [https://sylabs.io]

Setup Steps:

Install Nextflow

curl -s https://get.nextflow.io | bash

Install Singularity on CentOS Linux

sudo yum install singularity

Clone the repository

git clone --recursive https://github.com/JaneliaSciComp/multifish.git

Run setup to pull in external dependencies

./setup.sh

Launch the pipeline

./main.nf [arguments]
