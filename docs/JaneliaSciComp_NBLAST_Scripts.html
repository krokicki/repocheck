Setup Information

Prerequisites:

R installation [https://cran.cnr.berkeley.edu/]

Setup Steps:

Download the latest precompiled binaries

Download the latest precompiled binaries from https://github.com/JaneliaSciComp/NBLAST_Scripts/releases.

Decompress and copy files

Decompress and copy all files into `Fiji.app/plugins` directory.

Launch Fiji and create NBLAST database

Launch Fiji.

Run NBLAST database creation

Run `Plugins>build nblastdb`.

Launch Fiji and run NBLAST search

Launch Fiji.

Run NBLAST search

Run `Plugins>nblast search`.

Launch Fiji.

Run NBLAST score matrix generation

Run `Plugins>nblast scoremat nlfh`.
