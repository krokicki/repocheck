Setup Information

Prerequisites:

Miniconda installation is required. [https://conda.io/miniconda.html]

Setup Steps:

Add conda-forge channel to conda configuration

conda config --add channels conda-forge

Add flyem-forge channel to conda configuration

conda config --add channels flyem-forge

Create a new conda environment named 'neutube'

conda create neutube-python -n neutube

Activate the 'neutube' environment

source activate neutube

Install the package with setup script

python setup.py install
