Setup Steps:

Load the SymmetricFormats package

julia> using SymmetricFormats

Create a random 5x5 matrix

julia> A = rand(5,5)

Convert the matrix to a symmetric packed format (upper triangle)

julia> AP = SymmetricPacked(A)

Access the packed triangle of the symmetric matrix

julia> AP.tri

Get the size of the packed array

julia> packedsize(AP)
