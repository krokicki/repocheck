Setup Information

Prerequisites:

You need to have the AWS CLI installed and configured with proper AWS credentials. [https://aws.amazon.com/cli/]

Setup Steps:

Change to the VPC directory and install dependencies.

cd vpc

Install the npm packages.

npm install

Deploy the shared VPC using Serverless.

npm run sls -- deploy

Update the system and install necessary packages for the EC2 instance.

sudo yum -y update

Install the fuse-devel package.

sudo yum install -y fuse-devel

Format the volume as ext4.

sudo mkfs -t ext4 /dev/xvdb

Create a mount point for the scratch volume.

sudo mkdir /scratch_volume

Append the mount information to fstab.

sudo echo -e '/dev/xvdb /scratch_volume ext4 defaults 0 0' | sudo tee -a /etc/fstab

Mount all filesystems specified in fstab.

sudo mount -a

Stop the ECS service.

sudo stop ecs

Remove the ECS agent data to prepare for new configurations.

sudo rm -rf /var/lib/ecs/data/ecs_agent_data.json

Change to the alignment directory and install dependencies.

cd align

Install the npm packages for the alignment service.

npm install

Deploy the alignment service using Serverless.

npm run sls -- deploy -s dev

Change to the search directory and install dependencies.

cd search

Install the npm packages for the color depth search service.

npm install

Deploy the color depth search service using Serverless.

npm run sls -- deploy -s dev
