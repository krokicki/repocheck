Setup Information

Prerequisites:

Download and unzip the release file for the project [https://github.com/JaneliaSciComp/Zebrafish_96well_segmentation_mesure_FPC/releases/tag/zebrafish]

Setup Steps:

Launch FIJI and ensure it's updated

Launch FIJI

Copy the Zebra_96well.ijm macro to the FIJI plugins directory

cp Zebra_96well.ijm /Fiji.app/plugins/Macros/

Place the required .jar files into the FIJI plugins directory

cp Size_based_Noise_elimination.jar /Fiji.app/plugins/

Place the required .jar files into the FIJI plugins directory

cp Gamma_samewindow_noswing.jar /Fiji.app/plugins/

Place the required .jar files into the FIJI plugins directory

cp Local_contrast_thresholding.jar /Fiji.app/plugins/

Restart FIJI

Restart Fiji
