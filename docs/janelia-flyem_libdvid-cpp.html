Setup Information

Prerequisites:

Miniconda tool for conda installation [http://conda.pydata.org/miniconda.html]

Setup Steps:

Create a build directory and change into it

mkdir build

Run cmake to configure the build

cd build

Build the project

cmake ..

Compile the project

make

Install the project

make install

Create a conda environment and install libdvid-cpp

conda create -n -c flyem libdvid-cpp

Run the regression tests

make test
