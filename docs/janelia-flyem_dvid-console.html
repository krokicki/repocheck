Setup Information

Prerequisites:

Install and configure DVID [https://github.com/janelia-flyem/dvid]

Install node and npm [https://nodejs.org/]

Setup Steps:

Unpack the downloaded console tarball

tar -zxvf dvid-console-.tar.gz

Change directory to the clone of the repository

cd /into/the/repository/clone

Install all dependencies using npm

npm install

Start the console for testing

npm run start

Build the console

npm run build

Start the DVID server with the specified config

dvid -verbose serve /path/to/config.toml
