Setup Information

Prerequisites:

Docker or Singularity is required to run Maru [https://docs.docker.com/get-docker/]

Setup Steps:

Install Maru for Linux

curl -skL https://data.janelia.org/maru-linux | sudo tar -xz -C /usr/local/bin

Install Maru for MacOS

curl -skL https://data.janelia.org/maru-macos | sudo tar -xz -C /usr/local/bin

Initialize the project configuration

maru init

Build your project into a Docker container

maru build

Run the current version of your Docker container

maru run [args to app]

Run the Docker container using Singularity

maru singularity run [args to app]
