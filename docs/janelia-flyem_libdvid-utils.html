Setup Information

Prerequisites:

Miniconda needs to be installed for package management and environment handling [http://conda.pydata.org/miniconda.html]

Setup Steps:

Download and install Miniconda for Linux or Mac

wget https://repo.continuum.io/miniconda/Miniconda-latest-Linux-x86_64.sh

Run the Miniconda installer for Linux

bash Miniconda-latest-Linux-x86_64.sh

Download and install Miniconda for Mac

wget https://repo.continuum.io/miniconda/Miniconda-latest-MacOSX-x86_64.sh

Run the Miniconda installer for Mac

bash Miniconda-latest-MacOSX-x86_64.sh

Activate the conda environment

CONDA_ROOT=`conda info --root`

Source the conda root's activate script

source ${CONDA_ROOT}/bin/activate root

Create a new conda environment and install dvid-viewer

conda create -n CHOOSE_ENV_NAME -c flyem dvid-viewer

Create build directory and navigate into it

mkdir build

Change to the build directory

cd build

Run cmake to configure the build system with buildem

cmake .. -DBUILDEM_DIR=/user-defined/path/to/build/directory

Build the project with make using multiple processors

make -j num_processors

Run the test regressions

make test
