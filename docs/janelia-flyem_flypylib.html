Setup Information

Prerequisites:

Miniconda is required to create a conda environment and install packages. [http://conda.pydata.org/miniconda.html]

Setup Steps:

Download the Miniconda installer for Linux.

wget https://repo.continuum.io/miniconda/Miniconda3-latest-Linux-x86_64.sh

Run the Miniconda installer.

bash Miniconda3-latest-Linux-x86_64.sh

Activate the root conda environment.

CONDA_ROOT=`conda info --root`

Install flypylib, ipython, and matplotlib in a new conda environment.

conda create -n -c flyem-forge -c conda-forge flypylib ipython matplotlib
