Setup Information

Prerequisites:

A recent version of Matlab (at least 2012a), plus the image processing, signal processing, and statistics toolboxes []

The parallel computing toolbox if using a computer with many cores, or the distributed computing server or matlab compiler for a cluster []

Setup Steps:

Run ax1 to generate the sparse matrix of significant time-frequency pixels from the raw data file.

ax1('parameters.txt', 'rawdata.wav', '0')

Run ax2 to generate a list of bounding boxes from the previously created .ax file.

ax2('parameters.txt', {'rawdata-0.ax'}, 'rawdata')

Compile the code (this only needs to be done once).

./compile.sh

Batch process the files using the cluster.sh script.

./cluster.sh ./parameters.txt ./rawdata.wav 3
