Setup Information

Prerequisites:

Anaconda distribution of Python [https://www.anaconda.com/products/distribution]

Setup Steps:

Remove any existing kilosort environment if it exists

conda env remove -n kilosort

Create a new conda environment for Kilosort

conda create --name kilosort python=3.9

Activate the new environment

conda activate kilosort

Install kilosort with GUI support

python -m pip install kilosort[gui]

Uninstall CPU version of Pytorch (if installed)

pip uninstall torch

Install GPU version of Pytorch

conda install pytorch pytorch-cuda=11.8 -c pytorch -c nvidia

Run Kilosort GUI

python -m kilosort
