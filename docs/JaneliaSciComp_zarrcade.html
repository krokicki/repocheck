Setup Information

Prerequisites:

Install miniforge if you don't already have it. [https://github.com/conda-forge/miniforge]

Setup Steps:

Clone the repository

git clone https://github.com/JaneliaSciComp/zarrcade.git

Change directory to zarrcade

cd zarrcade

Initialize the conda environment

conda env create -f environment.yml

Activate the conda environment

conda activate zarrcade

Import images into Zarrcade

bin/import.py -d /root/data/dir -c mycollection

Run the Zarrcade web application

uvicorn zarrcade.serve:app --host 0.0.0.0 --reload
