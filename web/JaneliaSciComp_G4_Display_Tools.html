Setup Information

Prerequisites:

MATLAB version R2019b or newer []

Setup Steps:

Run the GUI to generate patterns

run G4_Pattern_Generator_gui.m

Run the function generator GUI for creating analog outputs and positions

run G4_Function_Generator.m

Connect to the G4 display system using PControl

run PControl_G4

Read data logged and acquired by the G4 display system into MATLAB

G4_TDMS_folder2struct('path/to/log/files')

Example script to run data analysis

run test_G4_Data_Analysis.m
