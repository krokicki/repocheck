Prerequisites:

Setup Steps:

Change directory to the folder containing MATLAB-SDK folder for SpikeGLX.

cd path/to/MATLAB-SDK-folder

Clone the Dromic repository from GitHub.

git clone https://github.com/JaneliaSciComp/dromic

Change directory to the dromic folder just created.

cd dromic

Install Dromic in MATLAB.

install_dromic

Run Dromic with fake data in MATLAB.

dromic('fake')
