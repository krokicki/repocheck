Setup Information

Prerequisites:

Java and maven must be installed. []

Setup Steps:

Clone the repo and enter its directory.

git clone https://github.com/PreibischLab/BigStitcher-Spark.git && cd BigStitcher-Spark

Run the included bash script to build the project and create executables for local use.

./install -t -m

Build the JAR for distribution.

mvn clean package -P fatjar

Run the resave command to process the stitching dataset.

./resave -x ~/SparkTest/Stitching/dataset.xml -xo ~/SparkTest/Stitching/dataset.xml

Run the stitching command to compute the transformations on the stitching dataset.

./stitching -x ~/SparkTest/Stitching/dataset.xml

Run the solver command on the stitching results.

./solver -x ~/SparkTest/Stitching/dataset.xml -s STITCHING

Run the match-interestpoints command to register each timepoint alignment individually.

./match-interestpoints -x ~/SparkTest/IP/dataset.xml -l beads -m FAST_ROTATION --clearCorrespondences

Run the solver with interestpoints on the specified dataset.

./solver -x ~/SparkTest/IP/dataset.xml -s IP -l beads
