Setup Information

Prerequisites:

CUDA.jl and SymmetricFormats.jl libraries are required for this project [https://github.com/JuliaGPU/CUDA.jl]

Ensure you have Julia installed [https://julialang.org/downloads/]

Setup Steps:

Load required packages

julia> using CUDA, SymmetricFormats, BatchedBLAS

Define the matrix size

julia> L = 4

Define the batch dimension

julia> N = 6

Create the reshaped array for the batched matrices

julia> A = reshape(1:L*L*N, L, L, N)

Create CuArray for storage of packed data

julia> SP = CuArray{Float64}(undef, packedsize(A), N)

Fill the CuArray with packed triangular matrices

julia> for i in 1:N SP[:,i] = SymmetricPacked(view(A,:,:,i), :U).tri end

Create a CuArray for the input vectors

julia> x = CuArray{Float64}(reshape(1:L*N, L, N))

Run the batched symmetric rank-k update operation

julia> batched_spr!('U', 1.0, x, SP)
