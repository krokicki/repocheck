Setup Information

Prerequisites:

Conda package manager for creating and managing environments [https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html]

Setup Steps:

Create a new Conda environment for the project

conda create -n exllsm-synapse

Update the Conda environment with dependencies from the environment file

conda env update -n exllsm-synapse -f environment.yml

Activate the newly created Conda environment

conda activate exllsm-synapse

Run the training script for the model

python ExLLSM_unet_v2_train.py

Plot the training progress results

python ExLLSM_unet_training_result_plot.py

Evaluate the model performance against ground truth data

python ExLLSM_unet_performance_evaluate.py
