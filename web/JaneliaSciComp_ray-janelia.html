Setup Information

Prerequisites:

A Conda environment with Ray installed [https://docs.ray.io/en/latest/ray-overview/installation.html]

Setup Steps:

Start a 20 slot cluster using the conda environment called ray-python

ray-janelia/ray-launch.sh -n 20 -e ray-python

Start a cluster with 20 CPU and 2 GPU slots on a GPU enabled queue

ray-janelia/ray-launch.sh -n 20 -e ray-python -b "-q gpu_queue -gpu num=2"

Create a cluster and run a python job in a single command

./ray-launch.sh -n 20 -e ray-python -p "/path/to/job.py --options"
