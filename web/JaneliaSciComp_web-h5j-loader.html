Setup Information

Prerequisites:

Node Package Manager (NPM) to install the module. [https://www.npmjs.com/get-npm]

Setup Steps:

Install the NPM module @janelia/web-h5j-loader

npm install @janelia/web-h5j-loader

Run tests for web-h5j-loader using Jest framework

npm run test

Run the conversion script using Docker for a TIFF stack to create an H5J file

docker run -v /tmp:/data janeliascicomp/flylight_tools:1.1.0 /app/scripts/cmd/convert.sh /data/example.tif /data/example.h5j /data/example.yml 0 1

Run the metadata script using Docker to extract metadata for the H5J file

docker run -v /tmp:/data janeliascicomp/flylight_tools:1.1.0 /app/scripts/cmd/h5jMetadata.sh /data/example.h5j /data/example.yml
