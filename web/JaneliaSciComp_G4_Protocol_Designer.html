Setup Information

Make sure MATLAB is installed and available in your PATH. [https://www.mathworks.com/products/matlab.html]

Setup Steps:

Open the settings file and check the configuration file path.

open G4_Protocol_Designer_Settings.m

Run the main script in MATLAB to start the designer.

run script.m

run run_conductor.m
