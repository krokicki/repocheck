Setup Information

Prerequisites:

Nextflow is required for running the pipeline. [https://www.nextflow.io/docs/latest/getstarted.html]

Singularity is required for running the pipeline. [https://sylabs.io/guides/3.7/admin-guide/installation.html]

Setup Steps:

Install Nextflow

curl -s https://get.nextflow.io | bash

Install Singularity on CentOS Linux

sudo yum install singularity

Run the Nextflow pipeline

nextflow run JaneliaSciComp/nf-n5-spark --indir /path/to/tiffs --outdir ./output -profile singularity
