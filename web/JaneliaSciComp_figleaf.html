Setup Steps:

Generate Python models from JSON schema for figshare private articles.

datamodel-codegen --input private_article_schema.json --input-file-type jsonschema --output priv_article_models.py

Get figshare metadata information.

python get_figshare_info.py -h

Convert CSV to JSON with compatibility checking for researcher metadata.

python ingest_researcher_metadata.py

POST a private article with metadata to figshare's server and publish it.

python create_and_publish.py

PUT a file on figshare associated with an existing article.

python publish_existing.py
