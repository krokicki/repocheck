Setup Information

Prerequisites:

Matlab R2013b or later (64-bit) []

Setup Steps:

Unzip the .zip file to a place of your choosing

unzip tmt.zip

In Matlab, go to File > Set Path...

matlab

Remove any paths for older versions of TMT

Navigate to the path settings in Matlab

Add TMT folder to Matlab path

Add the tmt-release_(whatever) folder using 'Add with Subfolders...'
